Descripción del grado

El Grado en Ingeniería Informática de la Universidad Francisco de Vitoria tiene como objetivo formar profesionales capaces de diseñar, desarrollar y mantener sistemas informáticos, combinando una sólida base teórica con una formación práctica.

Estructura del grado

El grado se organiza en diferentes cursos académicos que abarcan materias básicas, obligatorias y optativas.

Asignaturas

  • Fundamentos de Ingeniería Informática
  • Programación
  • Matemáticas
  • Estructuras de Datos
  • Sistemas Operativos
  • Bases de Datos

La única asignatura con una página específica en este sitio web es Fundamentos de Ingeniería Informática, tal y como se indica en el enunciado del trabajo.
